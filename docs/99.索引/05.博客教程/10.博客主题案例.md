## 代码块分组

::: warning

- `<code-block>`标签之间用空行隔开，否则可能会解析不出来。
- `<code-group>`标签 markdown 内容之间使用空行隔开，否则可能会解析不出来。

:::

**输入：**

````md
<code-group>
  <code-block title="YARN" active>
  ```bash
  yarn add vuepress-theme-vdoing -D
  ```
  </code-block>

  <code-block title="NPM">
  ```bash
  npm install vuepress-theme-vdoing -D
  ```
  </code-block>
</code-group>
````
