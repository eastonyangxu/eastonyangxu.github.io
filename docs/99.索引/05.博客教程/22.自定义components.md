输入：
::: theorem .vuepress/components/demo.vue

```vue -N
<style scoped lang="css">
h1 {
  color: red;
  font-size: 20px;
}
</style>

<template>
  <h1>这是我第一个自定义的components</h1>
</template>

<script>
export default {
  name: "demo",
};
</script>
```

使用 components：

```md -N
<demo/>
```
