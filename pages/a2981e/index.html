# 影调

  • 被摄体表面的不同亮度在黑白感光材料上所形成的阶调层次黑、灰、白以及处于这三个主要等级之间的过渡层次。
  • 依照调子的不同，摄影作品主要分为三个类型，即低调、中间调、高调。

# 光质

  • 硬光：光线直接照射，没有经过扩散，产生的阴影边缘清晰。硬光可以增强照片的对比度和立体感，常用于塑造强烈的视觉效果。 使用场景：时尚摄影、体育摄影、建筑摄影等需要强调细节和质感的场合。
  • 柔光：光线经过扩散，如通过云层或柔光箱，阴影边缘柔和。软光能够平滑肤色，减少阴影，适合温馨和浪漫的氛围。 使用场景：人像摄影、婚礼摄影、儿童摄影等需要营造柔和氛围

# 光位

  • 顺光：指光线从后方照射被摄物体，使得被摄物体正面充分照亮。这种光线可以呈现出清晰、明亮的效果，适合拍摄风景、人物等主体清晰的照片。
  • 侧光：指光线从侧面照射被摄物体，产生明暗对比，突出立体感和质感。适合拍摄人物、静物等，能够突出形状和纹理。
  • 逆光：指光线从被摄物体背后照射，使得主体呈现出剪影效果。适合拍摄轮廓、剪影等照片，能够营造出神秘、梦幻的氛围。
  • 顶光：指光线从上方照射被摄物体，适合用于强调被摄物体的表面纹理和细节，例如拍摄植物、建筑等。
  • 底光：指光线从下方照射被摄物体，通常用于营造神秘、恐怖的效果，适合拍摄特定主题的艺术照片。

# 景别

远取其势，近取其神

  • 远景：指距离观察者较远的景物，通常用于表现广阔的背景或远处的景色，如远山、远处的建筑等。在摄影中，远景可以用来营造开阔感和深远感，适合于风景摄影和城市风光摄影。
  • 全景：指包括整个画面的景物，通常用于展现整体环境或场景。在摄影中，全景可以用来展现完整的风景或场景，适合于旅游摄影和室内外全景摄影。
  • 中景：指距离观察者较近但不是最近的景物，通常用于突出主体或者连接远景和近景的过渡。在摄影中，中景可以用来突出主体并营造层次感，适合于人像摄影和静物摄影。
  • 近景：指距离观察者较近的景物，通常用于突出主体或者强调细节。在摄影中，近景可以用来突出细节并营造亲近感，适合于微距摄影和拍摄小型物品。
  • 特景：这个术语并不是常见的概念，可能是指在特定情境下需要突出的景物或特殊的景观。在摄影中，特景可能指特定的主题或特殊的场景，使用场景取决于具体的拍摄需求和创作意图。

# 视角

# 描述

  • 高视角：指摄影或摄像时，摄影机或摄像机位于被拍摄对象的上方，以一种俯视的角度来展现被拍摄对象。高视角可以使被拍摄对象显得较小，场景显得较大，常用于突出被拍摄对象的孤独、渺小或者无助感，也可以用于展现整体布局或者环境。
  • 平视角：指摄影或摄像时，摄影机或摄像机位于与被拍摄对象在同一水平线上的角度，使得画面呈现出一种与人眼视角相近的效果。平视角可以使观众更容易与画面中的人物或物体产生共鸣，常用于展现日常生活场景或者与观众产生情感共鸣。
  • 低视角：指摄影或摄像时，摄影机或摄像机位于被拍摄对象的下方，以一种仰视的角度来展现被拍摄对象。低视角可以使被拍摄对象显得更加庄严、威严或者强大，常用于突出被拍摄对象的力量、威严或者高大感，也可以用于突出被拍摄对象的英雄主义形象。

# 使用场景

  • 高视角：适合用于拍摄大场面、人群、城市景观等，也常用于拍摄小孩、动物或者其他显得渺小的对象，以突出环境的壮观或者被拍摄对象的无助感。
  • 平视角：适合用于拍摄日常生活场景、对话交流、人物情感表达等，能够使观众更容易产生共鸣。
  • 低视角：适合用于拍摄英雄、领袖、建筑物等，能够突出被拍摄对象的威严、力量或者高大感。
