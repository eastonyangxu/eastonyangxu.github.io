# list-style 属性演示

提示

list-style 属性是一个简写对属性集合，包括 list-style-type, list-style-image, 和 list-style-position。

以任何顺序接受一个，二个或者三个关键词。如果 list-style-type 和 list-style-image 的值同时出现，list-style-image 会覆盖 list-style-type 的样式。
