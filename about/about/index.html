# 📚Blog 之如何而来

在看一篇文档的时候，无意间发现域名中带有 GitHub 字符，于是好奇宝宝上线，去查找相关信息，终于发现了神奇的大陆，GitHub Pages 这一个免费的静态服务器 羊毛。然后去找开源文档项目，找到 vueress，这一下就一发不可收拾，开始了vuepress_template (opens new window) 的练习。

熟悉了VuePress1.x (opens new window)本项目基于 VuePress1.x后，去 GitHub 查找相关开源项目，找到了 vuepress-theme-vdoing 源码 (opens new window)| vuepress-theme-vdoing 文档 (opens new window)，发现vdoing这个主题很适合做为博客主题，而且自带的很多组件也很好用。在根据vdoing主题搭建博客时，又在文档里面发现了一个很漂亮的首页博客模板Young Kbt blog (opens new window)，该博客也是基于 vdoing 开发的，所以很多基本就是COPY直接用就可以啦 😄。

所以，本博客是基于 VuePress 一个 Vue 驱动的静态网站生成器，主题使用了 vdoing，页面样式借鉴了 Young Kbt blog 而来的。

# 📚Blog 之到哪儿去

本博客项目主要用于记录在开发学习中的笔记，分享一些技术知识。本人后端开发，但是前端项目又能很好的展示效果，所以借助于本博客，会记录前端知识学习中的点点滴滴。也会记录后端工作中常见的问题，及解决办法。可以通过导航栏查看本博客主要的一些知识点，有兴趣的小伙伴 👨‍🎓 可以互相学习讨论。如果你喜欢本博客欢迎到GitHub (opens new window)点个 Star，或者交换友链 ( •̀ ω •́ )✧
